# Trainee Evaluation Program

This Python program captures a trainee's full name and scores in Python Syntax, Problem Solving, and Mini Project. 
It calculates the total and average score, assigns a performance grade, and determines the trainee's completion status. 

The program includes input error checking to ensure that only valid numeric scores between 0 and 100 are accepted. 
It demonstrates core Python concepts used in real software development, including functions, loops, exception handling, conditionals, and formatted output.
